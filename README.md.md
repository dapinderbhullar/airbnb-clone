# Airbnb Clone

A full-stack Airbnb Clone built with FastAPI and Next.js.

## Features

- User Authentication (JWT)
- Guest & Host roles
- Add, Edit and Delete Properties
- Book Properties
- My Bookings
- Reviews & Ratings
- Search, Filter and Sort Listings
- Protected Routes
- Responsive UI

## Tech Stack

### Frontend
- Next.js
- TypeScript
- Tailwind CSS

### Backend
- FastAPI
- SQLAlchemy
- SQLite
- JWT Authentication

## Run Locally

### Backend

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
